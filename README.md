# Todo App

A fully responsive Todo App built with HTML, CSS, and JavaScript.

## 🚀 Features
- Add / Delete / Edit tasks
- Filter tasks (All / Active / Completed)
- Search functionality
- Priority system (Low / Medium / High)
- Drag & Drop reorder
- LocalStorage persistence

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)
